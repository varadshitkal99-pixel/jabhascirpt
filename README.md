# Java Mini Projects Collection

A collection of console-based Java applications demonstrating core programming concepts, object-oriented design, and problem-solving skills. Built with JDK 25.

## üéØ Projects Included

### 1. Hangman Game (`Hangman.java`)
**Description:** Classic word-guessing game where players try to guess a hidden word letter by letter.

**Features:**
- Random word selection
- Lives/attempts tracking
- Interactive console gameplay
- Win/lose conditions

**Concepts:** Strings, Arrays, Loops, Conditional Logic

---

### 2. Quiz Application (`quiz.java`)
**Description:** Normal Java quiz program with multiple-choice questions and score tracking.

**Features:**
- Multiple quiz questions
- Score calculation
- Immediate feedback on answers
- Final results display

**Concepts:** Arrays, Scanner Input, Conditional Statements

---

### 3. Banking System (`bank.java`)
**Description:** Simple banking application for managing accounts with deposits, withdrawals, and balance checks.

**Features:**
- Account creation
- Deposit and withdrawal operations
- Balance inquiry
- Transaction validation

**Concepts:** OOP, Classes & Objects, Encapsulation

---

### 4. Rock Paper Scissors (`sps.java`)
**Description:** Classic game against computer opponent with score tracking.

**Features:**
- Play against computer
- Random computer choice
- Score tracking
- Play multiple rounds

**Concepts:** Random number generation, Switch statements, Loops

---

### 5. Temperature Converter (`temoconvo.java`)
**Description:** Converts temperatures between Celsius, Fahrenheit, and Kelvin.

**Features:**
- Multiple conversion options
- Input validation
- Accurate calculations
- User-friendly interface

**Concepts:** Mathematical operations, Methods, User input handling

---

### 6. Food Cart System (`todih.java`)
**Description:** Food cart management system using ArrayList for menu and orders.

**Features:**
- Menu management
- Order tracking
- ArrayList implementation
- Dynamic data handling

**Concepts:** ArrayList, Collections, Data structures

---

## üì¶ How to Run

### Prerequisites
- Java Development Kit (JDK) 25 or higher installed
- Command line / Terminal access

### Steps

1. **Clone the repository**
```bash
git clone https://github.com/varadshitkal99-pixel/java-mini-projects.git
cd java-mini-projects
```

2. **Compile a project**
```bash
javac Hangman.java
```

3. **Run the project**
```bash
java Hangman
```

Repeat steps 2-3 for any project you want to run, replacing the filename accordingly.
